# API Overview

Versioned routes:

- `/api/v1/auth/**`
- `/api/v1/tenants/**`
- `/api/v1/users/**`
- `/api/v1/inventory/**`
- `/api/v1/consents/**`
- `/api/v1/requests/**`
- `/api/v1/audit-logs/**`
- `/api/v1/dashboard/**`
- `/api/v1/ai/**`

The API uses a consistent error envelope and JWT bearer authentication for protected routes.
